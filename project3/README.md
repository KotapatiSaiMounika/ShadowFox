# 🦷 Sakthi Dental Clinic Website

A full-stack dental clinic web application built to provide patients with an interactive and user-friendly online experience.

This project includes a modern React frontend, an Express backend, MongoDB database integration, and a simple AI-powered chatbot.

---

## 🚀 Features

### 🖥 Frontend (React + Vite)
- Responsive and modern UI
- Multi-page navigation (Home, About, Treatments, Contact, FAQ)
- Reusable React components
- Clean and structured design
- Integrated AI chatbot (keyword-based response system)

### 🔧 Backend (Node.js + Express)
- REST API for handling patient form submissions
- MongoDB database integration
- Data validation and structured storage
- Environment variable support using dotenv

### 🗄 Database (MongoDB)
Stores patient details such as:
- Customer Name
- Patient ID
- Mobile Number
- Selected Treatment
- Custom Message/Query

## 🤖 AI Chatbot

The chatbot is implemented using a keyword-based response system.

It analyzes user input and generates responses based on detected keywords related to:
- Treatments
- Appointments
- Clinic timings
- Contact information

This provides a lightweight and fast interactive assistant experience.
